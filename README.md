# Arctic-blocks-site

Legacy redirect-only site for the Arctic Blocks public pages. The canonical public website now lives in `frosty-floes-site`.

## Current behavior

- `index.html` redirects to the canonical Frosty Floes Blocks product page.
- `support.html` redirects to the canonical Frosty Floes Blocks support route.
- `privacy-policy.html` redirects to the canonical Frosty Floes Blocks privacy route.
- `app-ads.txt` remains at the root for ad verification compatibility.
